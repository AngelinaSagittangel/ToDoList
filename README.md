# Kanban Board
- Простая, но функциональная канбан-доска для управления задачами. Проект реализован на чистом JavaScript без использования сборщиков (Webpack, Vite), фреймворков и сторонних библиотек.

## Задача
Создать инструмент для визуализации рабочего процесса (To Do → In Progress → Done), который:

- Позволяет пользователю управлять задачами через интуитивный интерфейс.
- Сохраняет состояние приложения (задачи, их порядок и статус) между перезагрузками страницы.
- Работает быстро и плавно за счет грамотной работы с DOM без лишних перерисовок.

## Ключевые возможности
- Drag‑and‑Drop на нативном API: Перемещение задач между колонками реализовано через нативные события dragstart, dragend, dragover, drop. Никаких тяжелых сторонних библиотек.
- Сохранение состояния: Все данные сохраняются в LocalStorage в формате JSON. При обновлении страницы доска восстанавливается ровно в том виде, в котором её оставил пользователь.
- CRUD‑операции: Создание новых задач, удаление отдельных элементов и очистка колонок.
- Динамические счетчики: Количество задач в каждой колонке пересчитывается и обновляется в реальном времени.

## Стек технологий
- Core: Vanilla JavaScript (ES6+), HTML5, CSS3.
- API: HTML5 Drag and Drop API, Web Storage API (localStorage).
- DOM Manipulation: Нативные методы (insertBefore, querySelector, addEventListener).
